## About
~(유튜브 프리미엄 이용자들에게도 어거지로 광고를 보여주는)~ 스포티비 관련 채널들의 영상 속 광고를 자동으로 스킵하는 chrome extension

## 기능
- 스포티비 관련 채널들의 유튜브 영상을 볼 때 영상 속 광고가 끝나는 시점인 8초로 재생 시간을 이동시킵니다.
- 현재 "SPOTV", "스포일러" 채널에 적용
